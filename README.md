# GitHub Pages 法务页

这个目录用于承接 `个人电竞测算分析` 的法务静态页，可直接上传到 GitHub 仓库并开启 GitHub Pages。

## 目录

- `index.html`
- `privacy.html`
- `terms.html`

## 推荐仓库名

- `personal-esports-legal`
- `grind-legal`

## GitHub Pages 推荐链接结构

如果仓库名是 `personal-esports-legal`，Pages 开启后通常可访问：

- `https://<你的用户名>.github.io/personal-esports-legal/privacy.html`
- `https://<你的用户名>.github.io/personal-esports-legal/terms.html`

## 后续需要同步修改

上传并确认 Pages 生效后，需要把以下位置的法务链接改成新的 GitHub Pages 地址：

- `ArenaStayPro/ArenaStayPro/Services/AppConfiguration.swift`
- App Store Connect 的隐私政策链接
- 审核说明里的法务页链接
